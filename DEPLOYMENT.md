# Deployment checklist

Repository name:

`joshigunjan/EO-FM-Living-Atlas`

1. Upload all files in this folder to the repository root.
2. Go to **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Choose `main` and `/root`.
5. Wait for the Pages deployment to turn green in the **Actions** tab.
6. Open:

`https://joshigunjan.github.io/EO-FM-Living-Atlas/`

## After deployment

Check that:

- The catalogue table loads.
- Clicking a row opens details in the right-hand panel.
- Task filters work.
- Paper/code/weights links open original sources.
- Status labels say `Curated seed` or `Specialized seed`, not internal labels such as `seed_verified`.
