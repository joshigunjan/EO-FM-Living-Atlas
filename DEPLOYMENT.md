# Deployment checklist

1. Create or open the GitHub repository `joshigunjan/EO-FM-Living-Atlas`.
2. Upload the files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/root`.
6. Open:

   `https://joshigunjan.github.io/EO-FM-Living-Atlas/`

After future edits, GitHub Pages usually refreshes the site within a few minutes.
