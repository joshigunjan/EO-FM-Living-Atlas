Community intake

Do you have an EO foundation-model or benchmark paper to report?

Submit an original paper, arXiv link, DOI, GitHub repository, Hugging Face model, project page, or copied LinkedIn post text. New entries should enter as candidates and become public after review.

Recommended submission route

Use the GitHub issue form. The issue template asks for the paper link, code/weight links, modalities, architecture, downstream tasks, and openness.

Open submission form

Fields we need

  • Paper title and original paper link
  • Model or benchmark name
  • Model type: visual, multimodal, SAR, hyperspectral, vision-language, benchmark, dataset, agent
  • Modalities and sensors
  • Architecture family
  • Specific downstream tasks, not just “many tasks”
  • Code link, weights link, dataset link, and license if known
  • Why this should be included
