Community submissions

Do you know an EO foundation-model or benchmark paper that should be added?

Submit the original paper, arXiv link, DOI, GitHub repository, Hugging Face model, project page, or copied post text. The key requirement is that the entry can be traced back to primary sources.

Submit an entry

Use the GitHub issue form. It asks for the paper link, code and weight links, modalities, architecture, downstream tasks, and openness.

Open submission form

Information to include

  • Paper title and original paper link
  • Model or benchmark name
  • Model type: visual, multimodal, SAR, hyperspectral, vision-language, benchmark, dataset, agent, or generative
  • Modalities and sensors
  • Architecture family
  • Specific downstream tasks, not just “many tasks”
  • Code link, weights link, project page, dataset link, and license if known
  • Any caveat about availability or claims
